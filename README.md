Проект №14: Многоагентная Итеративная Система (LangGraph + Ollama)
📝 Описание проекта
Данный проект представляет собой автономного ИИ-агента, построенного на графовой архитектуре. В отличие от обычного чат-бота, эта система использует цикл самокоррекции, имитируя процесс работы реальной редакции: написание, критику и исправление текста.

⚙️ Технологический стек
Язык: Python 3.10+

Фреймворк: LangGraph (для управления логикой переходов и состоянием агента).

LLM: Ollama (локальная модель Llama 3). Это обеспечивает приватность данных и бесплатную работу без API-ключей.

Библиотеки: langchain-ollama, typing.

🏗️ Архитектура системы
Программа состоит из трех ключевых узлов (ролей), связанных в циклическую структуру:

Writer (Писатель): Генерирует первичный контент на основе заданного пользователем задания (task).

Critic (Критик): Анализирует полученный текст. Если текст соответствует качеству, он выдает сигнал ГОТОВО. Если нет — формирует список конкретных правок.

Editor (Редактор): Принимает черновик и замечания критика, после чего переписывает текст, улучшая его.

Логика переходов:
Система работает итеративно. Она будет отправлять текст на доработку до тех пор, пока Критик не будет удовлетворен, либо пока не будет достигнут лимит в 3 итерации (защита от бесконечного цикла).

🚀 Как это работает (алгоритм)
Пользователь вводит тему в task_input.

Граф запускает узел writer.

Результат передается в critic.

Функция should_continue проверяет ответ критика:

Если найдено слово "ГОТОВО" — переход в END.

Если есть правки — переход в editor и возврат к критику.

Финальный, «отшлифованный» текст сохраняется в файл report.txt.

📊 Результат работы
В ходе тестов агент успешно продемонстрировал способность:

Самостоятельно исправлять структуру текста.

Придерживаться заданного языка (русского).

Аргументировать тезисы, используя знания модели.
